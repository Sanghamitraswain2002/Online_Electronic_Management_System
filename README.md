# Online Electronic Management System

## 📌 Description

This project is an Online Electronic Management System developed using Core Java, Advanced Java, JSP, and Oracle Database. It helps manage electronic products, users, and orders in a simple and efficient way.

## 🚀 Technologies Used

* Core Java
* Advanced Java (JSP, Servlets)
* Oracle Database

## 👤 Modules

### 🔐 Admin Module

* Admin can register and login using username and password
* Add new products
* Edit product details
* Delete products

### 🛒 Customer/User Module

* User can register and login
* View available products
* Buy products

## 💡 Features

* Secure login system
* Product management
* User-friendly interface
* Database integration

## 📂 Project Structure

* JSP Pages
* Java Classes
* Database (Oracle)

## ▶️ How to Run

1. Import project into Eclipse/IDE
2. Configure Oracle Database
3. Run on server (Tomcat)
4. Open in browser
